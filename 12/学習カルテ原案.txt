問1：回帰分析の目的として最も適切なものはどれですか。
  1. 2つの量の関係から、目的変数を予測するモデル（式）を作る
  2. 平均が基準値と違うかをp値で判断する
  3. データをカテゴリ別に集計して割合を出す
  4. 2つの量が同じ分布かどうかを検定する
正解：1
解説：回帰は「関係を式で表し、予測に使う」ための方法です。検定（p値で判断）とは目的が異なります。

問2：散布図（scatter plot）の説明として最も適切なものはどれですか。
  1. 2つの数値の組を点として描き、関係を見える化するグラフ
  2. 1つの数値の回数（度数）を棒で表すグラフ
  3. 時間の変化を折れ線で表すグラフ
  4. 割合の内訳を円で表すグラフ
正解：1
解説：散布図は「(x, y)の組」を点で示し、右上がり・右下がり・関係なしなどを読み取ります。

問3：回帰分析で使う変数の呼び方として適切なのはどれですか。
（例：x＝バーガー注文数、y＝ポテト注文数）
  1. xは説明変数、yは目的変数
  2. xは目的変数、yは説明変数
  3. xは標本、yは母集団
  4. xは帰無仮説、yは対立仮説
正解：1
解説：回帰では、原因側・入力側を説明変数x、予測したい側を目的変数yと呼びます。

問4：散布図の読み方として「右上がり」に最も近い説明はどれですか。
  1. xが大きいほどyも大きくなる傾向がある
  2. xが大きいほどyが小さくなる傾向がある
  3. xの値に関係なくyが一定になる
  4. xとyの関係を判断できない
正解：1
解説：右上がりは「増える傾向」を意味します（強さは点のまとまり具合で変わります）。

問5：回帰直線の役割として最も適切なものはどれですか。
  1. 点の雲（データ）を一本の線で要約し、予測に使える形にする
  2. 外れ値を自動で削除して、データをきれいにする
  3. xとyを同じ平均・分散に変換する
  4. 2つのデータの平均との差が偶然かどうかを判断する
正解：1
解説：回帰直線は点の傾向を要約し、xからyを予測するための道具です。

問6：回帰式の形として適切なものはどれですか。
  1. ŷ = ax + b
  2. p = 2*(1 - NORM.S.DIST(|z|, TRUE))
  3. Z = (x̄ - μ0) / (σ/√n)
  4. t = (x̄ - μ0) / (s/√n) を使って棄却域を見る
正解：1
解説：単回帰の基本形は ŷ=ax+b です。ŷは予測値（ハット付き）です。

問7：回帰式 ŷ=ax+b における「傾きa」の意味として最も適切なのはどれですか。
  1. xが1増えたとき、y（予測値）が平均でどれくらい増えるか
  2. xが0のときのy（予測値）
  3. 予測が当たる確率
  4. 外れ値の個数
正解：1
解説：傾きaは「増え方」を表します。例：a=0.6なら、xが10増えるとŷは約6増える、のように解釈します。

問8：R²（決定係数）の説明として最も適切なものはどれですか。
  1. 回帰直線でyのばらつきのどれくらいが説明できたかの目安（正解率ではない）
  2. 回帰直線が正しい確率
  3. 外れ値がある確率
  4. yの平均がμ0と等しい確率
正解：1
解説：R²は「当てはまりの目安」です。1に近いほど点が線に近い傾向ですが、未来が必ず当たる保証ではありません。

問9：「相関≠因果」の意味として最も適切なのはどれですか。
  1. 一緒に増減する関係が見えても、原因が確定するとは限らない
  2. 回帰式が作れれば、原因も必ず特定できる
  3. R²が高ければ、必ず因果関係がある
  4. 外れ値があると相関がゼロになる
正解：1
解説：回帰・相関は「関係」を示しますが、因果（原因）は別要因（来客数、キャンペーン等）で説明できる場合があります。

問10：cleanデータとoutlierデータを使い分ける理由として最も適切なのはどれですか。
  1. cleanで回帰の基本を成功させ、outlierで外れ値が線・R²・予測をどう変えるか学ぶため
  2. cleanは検定用、outlierは確率分布用だから
  3. cleanは平均を計算しないため、outlierで平均を出すため
  4. outlierは必ず削除してはいけないので、cleanだけを使うため
正解：1
解説：cleanは「回帰の型」を学ぶために読みやすく、outlierは「現実のズレ」を体験して注意点（残差・外れ値）を学ぶために使います。
