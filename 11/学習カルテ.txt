問1：1標本t検定の目的として最も適切なものはどれですか。
  1. 標本の平均が基準値（例：135g）と違うと言ってよいかを判断する
  2. 2つのグループの平均が同じかどうかを判断する
  3. データの最大値と最小値を求める
  4. 回帰直線を求めて予測する
正解：1
解説：1標本t検定は、1つの標本の平均が「基準値」と異なると言えるかを、ばらつきを考慮して判断します。

問2：ポテトの検定で立てる帰無仮説 H0 として最も適切なものはどれですか。
  1. μ = 135
  2. μ < 135
  3. μ ≠ 135
  4. σ = 135
正解：1
解説：帰無仮説は「いつも通り（基準どおり）」を置くため、平均が135gである（μ=135）がH0です。

問3：母標準偏差 σ が現実には分からないことが多い理由として最も適切なものはどれですか。
  1. 店全体（母集団）をすべて測ることは現実的に難しいから
  2. σはExcelでは計算できないから
  3. σは日によって必ず0になるから
  4. σは平均と同じ値になる決まりだから
正解：1
解説：σは「母集団全体のばらつき」なので、本来は母集団全体を調べる必要があり、現実には難しいため未知として扱います。

問4：母標準偏差 σ が未知のとき、代わりに使う量として最も適切なものはどれですか。
  1. 標本標準偏差 s
  2. 標本平均 x̄
  3. 有意水準 α
  4. p値
正解：1
解説：σが未知の場合、標本から計算した標本標準偏差 s をばらつきの見積もりとして使います。

問5：1標本t検定で使うt値の式として最も適切なものはどれですか。（μ0は基準値）
  1. t = (x̄ − μ0) / (s / √n)
  2. t = (μ0 − x̄) / (σ / √n)
  3. t = (x̄ − μ0) / (s − √n)
  4. t = (x̄ + μ0) / (s / √n)
正解：1
解説：平均との差を「標本平均のゆれ（s/√n）」で割って位置（t）を作ります。σが未知なのでsを使います。

問6：t値が表しているものとして最も適切な説明はどれですか。
  1. 「平均との差」が標本平均のゆれ（s/√n）の何倍かを表す値
  2. 「平均との差」がそのままの大きさで表される値
  3. 「平均との差」が有意水準の何倍かを表す値
  4. 「平均との差」が標本標準偏差の何倍かを表す値
正解：1
解説：tは「ずれ ÷ 平均のゆれ」であり、分布上の位置（珍しさの判断のための尺度）です。

問7：Excelで標本標準偏差（不偏標準偏差）を求める関数として最も適切なものはどれですか。
  1. STDEV.S
  2. STDEV.P
  3. AVERAGE
  4. NORM.S.DIST
正解：1
解説：STDEV.Sは標本標準偏差（分母がn−1）を返します。STDEV.Pは母集団全体がある前提です。

問8：自由度 df の値として最も適切なものはどれですか。（標本数 n=30）
  1. df = n − 1 = 29
  2. df = n = 30
  3. df = n + 1 = 31
  4. df = 2n = 60
正解：1
解説：1標本t検定では自由度は df = n−1 を用います（標本からsを推定するため）。

問9：p値の意味として最も適切なものはどれですか。
  1. H0が正しいと仮定したとき、今のt値以上に極端な結果が出る確率
  2. 帰無仮説H0が正しい確率
  3. 標本平均が基準値と一致する確率
  4. 有意水準そのもの
正解：1
解説：p値は「H0のもとでの珍しさ」です。H0が正しい確率ではありません。

問10：有意水準0.05の両側検定で、p値=0.03だった。判断として最も適切なものはどれですか。
  1. p値 < 0.05なのでH0を棄却する
  2. p値 ≥ 0.05なのでH0を棄却できない
  3. p値 < 0.05なのでH0が正しいと確定する
  4. p値 = 0.03なので平均は必ず135gより小さい
正解：1
解説：p値が有意水準より小さければ「珍しすぎる」のでH0を棄却します。ただし棄却＝確定ではありません。
