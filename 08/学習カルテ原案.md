【第7回 小テスト：平均の性質】

問1：30個のデータから計算した平均について、一般に正しい説明はどれですか。
  1. 毎回必ず同じ値になる
  2. 元データよりも必ず大きな値になる
  3. 計算するたびに、少しずつ異なる値になることがある
  4. 外れ値の影響を完全に受けなくなる
正解：3
解説：平均は1回の結果であり、条件が同じでも毎回完全に同じ値になるとは限らない。

問2：平均を1回だけ計算した結果について、最も適切な考え方はどれですか。
  1. それ自体が1つの観測結果である
  2. それ以上分析する必要はない
  3. 常に真の値を正確に表す
  4. データとは別物として扱う
正解：1
解説：平均も1つの結果であり、他の値と同様に分析対象となる。

問3：同じ条件で平均を何回も計算し、それらを集めたときにできるものは何ですか。
  1. 代表値
  2. 母集団
  3. 分布
  4. 外れ値
正解：3
解説：複数の平均を集めると、平均の値の分布ができる。

問4：元データの分布と、平均を集めた分布を比べたとき、一般に成り立つ性質はどれですか。
  1. 平均の分布のほうが、広がりが小さい
  2. 平均の分布のほうが、必ず左右に偏る
  3. 両者の分布の形は必ず同じになる
  4. 平均の分布のほうが、外れ値が増える
正解：1
解説：平均は複数の値をまとめた結果なので、ばらつきが小さくなる。

問5：「データ数を増やすほど、平均との差が小さくなり、平均が安定していく」
この性質を表す名前はどれですか。
  1. 中心極限定理
  2. 分散の定義
  3. 大数の定理
  4. 正規分布の性質
正解：3
解説：平均が安定していく性質を表すのが大数の定理である。

問6：中心極限定理について、正しい説明はどれですか。
  1. 元データが正規分布である場合にのみ成り立つ
  2. 平均を何回も集めると、その分布の形が似てくる
  3. 平均を1回計算すれば正規分布になる
  4. 外れ値を除くと平均は変化しなくなる
正解：2
解説：中心極限定理は、平均の分布の形に関する性質を述べたものである。

問7：大数の定理と中心極限定理の違いとして、最も適切なものはどれですか。
  1. 両者は同じ内容を別の言葉で表している
  2. 大数の定理は平均の安定、中心極限定理は平均の分布の形を扱う
  3. 中心極限定理は平均との差のみを説明する
  4. 大数の定理は分布の形を決定する
正解：2
解説：LLNは「平均がどこに近づくか」、CLTは「平均の集まりがどんな形になるか」を扱う。

問8：平均を何回も計算して得られた値が、特定の範囲に集中して現れる理由として最も適切なものはどれですか。
  1. 偶然の影響が強くなるため
  2. 個々のズレが足し合わされるため
  3. 正と負のズレが打ち消し合うため
  4. 外れ値が自動的に除去されるため
正解：3
解説：平均は複数の値をまとめるため、ズレが相殺されやすい。

問9：平均の分布が利用できるようになることで、可能になる判断として適切なものはどれですか。
  1. すべてのデータが正しいかどうかの判定
  2. ある平均値がどの程度珍しいかの判断
  3. 個々のデータの正確さの保証
  4. データ収集の必要性の否定
正解：2
解説：平均の分布を使うことで、その値がどれくらい外れているかを判断できる。

問10：次の文の（　）に入る最も適切な組み合わせはどれですか。

「平均は1回の結果では（　）ことがあるが、集めてみると（　）な性質が見えてくる。」

  1. 固定される／不安定
  2. 揺れる／規則的
  3. 正確になる／ランダム
  4. 一定になる／偶然的
正解：2
解説：平均は揺れるが、集めることで規則性が現れる。
