問1：単回帰で「予測値」を表す記号として最も適切なものはどれですか。
  1. ŷ（yハット）
  2. y
  3. e
  4. x
正解：1
ŷ（予測値）は回帰式から計算される「予測されたy」です。yは実測値、eは残差、xは説明変数です。

問2：散布図の説明として最も適切なものはどれですか。
  1. 2つの数値の関係を点の集まりで表すグラフ
  2. 1つの数値の分布（形）を棒で表すグラフ
  3. 順位の変化を線で表すグラフ
  4. 割合を円で表すグラフ
正解：1
散布図は「(x, y) のペア」を点で表し、関係（右上がり・右下がり・関係なし）や外れ値を読み取ります。

問3：回帰直線の目的として最も適切なものはどれですか。
  1. 点の雲を1本の線で要約し、予測に使う
  2. 外れ値を必ず取り除く
  3. 2つの変数の単位を同じにする
  4. すべての点を必ず通る線を作る
正解：1
回帰直線は「傾向を要約して予測に使う」ための線で、すべての点を通る必要はありません。

問4：回帰式 ŷ = ax + b における「傾き a」の意味として最も適切なものはどれですか。
  1. xが1増えると、ŷが平均でどれくらい増えるか
  2. xが0のときのyの最大値
  3. yが1増えると、xがどれくらい増えるか
  4. 点の数（データ数）
正解：1
傾きaは「増え方」を表します。xが1増えたときに予測値ŷがどの程度増えるかの目安です。

問5：回帰式 ŷ = ax + b における「切片 b」の説明として最も適切なものはどれですか。
  1. x=0のときの予測値（ただし意味がない場合もある）
  2. 点と点の間の距離
  3. yの平均値そのもの
  4. 外れ値の位置
正解：1
切片bはx=0のときの予測値ですが、x=0が現実にありえない設定では解釈が難しいことがあります。

問6：R²の説明として最も適切なものはどれですか。
  1. 回帰直線で説明できる割合の目安（正解率ではない）
  2. 回帰式が必ず当たる確率
  3. 外れ値がある確率
  4. yの平均値
正解：1
R²は「yのばらつきをどれだけ線で説明できたか」の目安です。未来の予測が保証されるわけではありません。

問7：残差 e の定義として最も適切なものはどれですか。
  1. e = y − ŷ（実測−予測）
  2. e = ŷ − y（予測−実測）
  3. e = x − y
  4. e = ax + b
正解：1
残差は「実測値と予測値のズレ」です。線より上なら＋、下なら−になります。

問8：残差をそのまま足した「残差の合計」が基準になりにくい理由として最も適切なものはどれですか。
  1. ＋と−が打ち消し合って小さくなることがある
  2. 残差は必ず0になるから
  3. 残差は必ず正の値になるから
  4. 残差は単位を持たないから
正解：1
残差は正負があるので合計すると相殺します。そこで二乗してから足し合わせます。

問9：SSE（残差平方和）の式として最も適切なものはどれですか。
  1. Σ (y − ŷ)²
  2. Σ (y + ŷ)²
  3. Σ (x − y)²
  4. Σ (ax + b)
正解：1
SSEは「ズレ²の合計」です。ズレの大きさをまとめて評価する指標として使います。

問10：最小二乗法で決まる回帰直線の説明として最も適切なものはどれですか。
  1. SSEが最小になる直線
  2. すべての点を必ず通る直線
  3. R²が必ず1になる直線
  4. 外れ値だけを通る直線
正解：1
最小二乗法では「残差平方和SSEが最小」になる直線を回帰直線として選びます。外れ値があると二乗の影響で線が引っ張られることもあります。
