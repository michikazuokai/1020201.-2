問1：次のうち「母集団」の説明として最も適切なものはどれですか。
  1. 抽出した30個のデータ（標本）
  2. 標本から計算した平均値
  3. 観測できないことが多い「全体」の集まり
  4. 標本平均を何回も集めた分布
正解：3
解説：母集団は「本当は知りたい全体」です。現実には全部を調べられないため、標本を使って考えます。

問2：次のうち「標本」の説明として最も適切なものはどれですか。
  1. 母集団そのもの（全データ）
  2. 母集団から一部として取り出した限られたデータ
  3. 母集団の平均（母平均）
  4. 標準偏差が0になるように加工したデータ
正解：2
解説：標本は母集団の一部です。母集団全体を調べられないため、標本を取り出して計算します。

問3：標本平均が「毎回同じ値にならない」主な理由として最も適切なものはどれですか。
  1. 平均の計算式が日によって変わるから
  2. 取り出される標本（データの組）が毎回違うから
  3. 母集団の平均は常に一定ではないから
  4. 標本サイズnが大きいほど誤差が必ず増えるから
正解：2
解説：標本が変われば含まれるデータも変わるため、標本平均は少しずつ揺れます。これは計算ミスではなく性質です。

問4：「点推定」の説明として最も適切なものはどれですか。
  1. 母集団の値を範囲（下限〜上限）で表すこと
  2. 標本から母集団の値を1つの数で推測すること
  3. 標本を増やして母集団を完全に再現すること
  4. 標本平均を必ず母平均と同じにする操作
正解：2
解説：点推定は、標本から母集団の母数（例：母平均）を「1つの数」で推測する方法です。

問5：次のうち「区間推定」の説明として最も適切なものはどれですか。
  1. 母集団の値を1つの数に決め打ちする方法
  2. 標本を並べ替えて平均が変わらないようにする方法
  3. 母集団の値を“ありそうな範囲”として表す方法
  4. 母集団の全データを調べて範囲を確定する方法
正解：3
解説：区間推定は、点推定に「幅（不確かさ）」を付けて、母数が入りそうな範囲で表現します。

問6：区間推定で「区間の広さ（幅）」に強く関係する量として最も適切なものはどれですか。
  1. 標本の中央値
  2. 標本の最大値
  3. 標本標準偏差（ばらつき）
  4. データの並び順（昇順か降順か）
正解：3
解説：標本のばらつきが大きいほど、推定区間は広くなりやすい（不確かさが大きい）と考えます。

問7：次のうち、標本標準偏差が大きいときの推定区間について最も適切なものはどれですか。
  1. 推定区間は狭くなる（確実になる）
  2. 推定区間は広くなる（不確かになる）
  3. 推定区間は必ず0になる
  4. 推定区間の中心（平均）が必ず変わらない
正解：2
解説：ばらつきが大きい標本ほど、母平均の位置を絞り込みにくくなるため、区間は広くなりやすいです。

問8：次の文として最も適切なものはどれですか。
  1. 推定区間は、母平均が必ずその中に入ることを保証する
  2. 推定区間は、標本平均が必ずその中に入ることを保証する
  3. 推定区間は、母平均がその範囲に“入りそうだ”と言える形にする
  4. 推定区間は、母集団を直接測定した結果そのものである
正解：3
解説：「必ず入る」と断言しないのが重要です。区間推定は不確かさを含めた表現（“ありそうな範囲”）です。

問9：推測統計の考え方として最も適切なものはどれですか。
  1. 標本の結果だけを見て、母集団のことは考えない
  2. 母集団を全部測定してから判断する
  3. 標本の結果を使って、見えない母集団について考える
  4. データの揺れを無視して、平均だけで結論を出す
正解：3
解説：推測統計は「標本から母集団を考える」方法です。標本の揺れを前提に、根拠を持って判断します。

問10：次のうち、点推定の限界として最も適切なものはどれですか。
  1. 平均が計算できない
  2. データ数が増えるほど必ず外れる
  3. どれくらいズレているか（不確かさ）が分からない
  4. 母集団の全データが必要になる
正解：3
解説：点推定は1つの数で表せて分かりやすい一方、「どれくらいズレそうか」が分からないため、区間推定で幅を考えます。
